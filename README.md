# Vibes OBS Template

Professional recording setup for tutorial videos.

## Installation

1. Download vibes-obs-complete-template.zip
2. Extract the ZIP file
3. Run the installer:
   - Windows: Double-click install-complete-windows.bat
   - Mac: Terminal → sh install-complete-mac.sh
4. Open OBS Studio
5. Select Scene Collection → vibes-tutorial-template
6. Select Profile → vibes-recording-profile
7. Right-click "Desktop" source → Properties → Select your monitor
8. Start Recording!

Your videos will be saved in the Videos folder.

## Features

- Auto-zoom on clicks (no hotkeys!)
- Pre-configured scenes
- Professional quality settings
- Vertical video support (with Aitum Vertical)
- Ready in 2 minutes

## Requirements

- OBS Studio 30+
- Windows 10+ or macOS 10.15+
- Optional: Aitum Vertical plugin for vertical videos
